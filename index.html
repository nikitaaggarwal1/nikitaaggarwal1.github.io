Steam Engine Turbine

This is a Steam Turbine Engine designed during my second year’s Thermodynamics course. The turbine buckets were 3D printed to capture maximum wind possible. The turbine coverts thermal energy to mechanical and the motor converts mechanical energy to electrical energy. Used thermodynamic laws to meet design specification such as voltage and rotational turbine speed, under a restricted budget and time frame.

Ansys Software - Stress Distribution

A 2-Dimensional triangular element with a semi circle cut out from the bottom edge was defined using Young’s elastic modulus and Poisson’s ratio. The ‘plane 183’ element was given a boundary condition by fixing the left side and applying a distributed load on the right side and recording the plain stress distribution. Progressive accuracy of the stress distribution’s x component was achieved by increasing the mesh elements to obtain a more finer mesh. The iterative process of the mesh refinement assisted in revealing local stress concentration patterns in an area which is close to the load application, until eventually convergence is reached.

Digital Dashboard

The project “Digital Dashboard” is designed for a mini car or a truck, depending on the user’s input. The dashboard consists of displaying the speed, RPM, average fuel economy and fuel consumption, a compass and a range, regarding the type of vehicle. The code for digital Dashboard was implemented using Processing and logical commands such as loops or if- else condition statements were utilized in order to prompt the user for an input, calculate direction, fuel economy, current speed, or plot fuel consumption graph.

Sensory Robot

This is a sensory robot, which functions based on detecting an object via light sensitivity. The code was implemented in Processing and the prototypes were constructed in SolidWorks. The design process was followed such a defining the problem, gaining clarity or background information, confirming the exact problem by using all relevant facts and the testing stage. During the testing stage, a scoring - grid method was used in order to evaluate the degree of success for each category such as the design specification, time taken to perform the required tasks, aesthetic, reliability and functionality. Work and energy calculations were also performed in order for the sensory robot to be able to pick up the block of wood which was a design specification.
